    • Web Bots

    Open Source: Worker bees of the internet Web robots From Open Source forums: Web Robots: The worker bees of the Internet. Traverse internet to extract information. Used both for legitimate and malicious purposes. When we think robots, we often think terminator, or drones sometimes associated by a dystopian future where they rule humans. Web robots however are scripts. Scripts that fetch, analyse and file web content at very fast rate. A fun fact is that almost half of web traffic is cased and enabled by web robots that are visiting different websites. Referred to as the worker bees of the internet. Anyone can create web bots, you achieve this by using source scripting e.g. PHP, c etc. You can programme bots to perform certain tasks. Web robots can make use of instant messaging and other use interfaces to communicate with one another. They can reply, comment, change and alter information as well. This is why we get a lot of Are you a robot tests on websites we visit these days. Certain web bots can be trained to perform specific functions therefore you get types of bots for different subjects. Spider Web bots link and gather information that have been explored, Trading bots are commercial, they search for keywords, or a specific product or service and provides user with best results. Media bots provide updates, censors and are widely used for online messengers In terms of maliciousness, there are spam, which spread unwanted or unnecessary advertisements/pop ups, known for finding email addresses and spamming them. Hacker bots search and destroy, finding vulnerable websites to be exploited etc. You should be very careful if you are trying to build web bots, take into consideration what is moral and immoral. For More steps and information visit: Web robots There are advantages and disadvantages of having Web bots. Some of the advantages are that they perform t a faster rate than humans and are reliable and efficient. Some disadvantages are that they cause traffic, exploit vulnerable websites and spam.
