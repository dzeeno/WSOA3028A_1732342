    Aira Aira Accessible Rich Internet Applications. (AIRA) Was created after HTML4 and is a set of attributes that define ways to make web content more accessible to people with disabilities. Usually with Java Script. It does things like update live, present widgets, etc. Aira is active within landmark elements. These landmark elements in html5 are main, header and nav.
